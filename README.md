## Requirements

- Modern web browser
- JavaScript enabled
- Audio output device (for notifications)

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Start your first Pomodoro session!

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
